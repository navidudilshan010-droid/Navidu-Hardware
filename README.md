# Navidu Hardware Website

A premium "Sales" focused website for Navidu Hardware, built with HTML, CSS, JavaScript, and Tailwind CSS.

## How to Run
Since this is a static website with no Node.js dependency (as requested), you can simply:

1.  **Double-click** on `index.html` to open it in your browser.
2.  Navigate through Home, Products, and Contact pages.

## Features
-   **Modern Design**: Using Tailwind CSS for a professional, industrial look.
-   **Product Catalog**: `products.html` features a dynamic product grid with properly working Category Filters (Tools, Paints, etc.) and Search.
-   **Mobile Responsive**: Works perfectly on phones with a mobile menu.
-   **WhatsApp Integration**: "Inquiry" buttons on products link directly to WhatsApp with a pre-filled message.

## Technologies Used
-   HTML5
-   Tailwind CSS (via CDN)
-   Vanilla JavaScript
-   Font Awesome (Icons)
-   Google Fonts (Inter)

## customization
To add more products, edit the `products` array in `assets/js/main.js`.
